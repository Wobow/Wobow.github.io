Réalisations

Site web Owl

Site Web Owl

Date:  Juillet 2016

Categorie:  Développement Web

Client:  Owl

Visiter le site

Création d'un site web permettant aux utilisateurs de rechercher des événements musicaux près de chez eux selon un certain nombre de critères.
Le site est réalisé avec AngularJS, HTML5 et CSS3 en front-end, et en PHP avec Phalcon en back-end avec une API RESTful.

La base de données dans laquelles sont référencés les événéments est une base MongoDB. Un moteur de recherche Elasticsearch a été implémenté pour réaliser des recherches plus efficaces dans la base.
La recherche permet de filtrer les événements par ville, style de musique ou salle de spectacle, et d'affiner les résultats avec des mots clés.

Site Owl Pro

Site Owl Pro

Date:  Avril 2016

Categorie:  Développement Web

Client:  Owl

Visiter le site

Ce site web permet aux promoteurs et organisateurs de référencer leur événements sur la plateforme Owl. Ceux-ci seront alors visible sur l'application mobile et le site web. Ils s'authentifie grâce à un système de token sécurisé.

Réalisé avec AngularJS en front-end et Phalcon PHP en back-end, la base de données est une base MongoDB. Les promoteurs référencent leurs événements à travers un formulaire permettant d'avoir un maximum d'informations : lieu, styles de musique, artistes présents, prix, etc.
Une page recense également toutes les statistiques relatives à l'événement selectionné
Ils peuvent également co-organiser des événements avec d'autres promoteurs, et gérer un profil visible par les utilisateurs.

ID3 Biometrics

ID3 Biometrics

Date:  2015

Categorie:  Développement mobile

Client:  ID3 Technologies

Création de deux applications android permettant de mettre en pratique les algorithmes de reconnaissances d'empreintes digitales et faciales.

Les applications ont été développées en Java Android natif. L'une était connectée à un lecteur d'empreintes digitales Futhronic, et permettait la sauvegarde en base de donnée locale de 10 doigts et de plusieurs individus. Une fonction d'identification était également mise à disposition, permettant à un utilisateur de passer son doigt, et de valider la correspondance avec son profil en base.
L'application de reconnaissance faciale fonctionne sur le même principe, avec l'intégration de la caméra, et d'un tracker facial en real-time sur le flux vidéo.
